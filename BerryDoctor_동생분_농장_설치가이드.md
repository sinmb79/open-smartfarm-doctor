# 딸기박사 — 파일럿 농가 설치 가이드

> **대상**: 비닐하우스 3동 × 200평 / 설향 / 토경재배 / 반자동 관비
> **목표**: 병충해 조기 발견 → 비용 절감 → 광량 관리 → 침수 대응
> **예산**: 단계별 0원 ~ 50만원

---

## 1. 농장 현황 요약

| 항목 | 내용 |
|------|------|
| 하우스 | 비닐하우스 3동, 각 200평 (660㎡) |
| 품종 | 설향 (Seolhyang) |
| 재배 방식 | 토경재배 |
| 양액 관리 | 반자동 (관비기 있음 추정) |

| 요소 | 적정 범위 | 위험 수치 |
|------|----------|----------|
| 주간 온도 | 18~25°C | >30°C 고온장해, <5°C 동해 |

| CO2 | 600~1,000 ppm | <400 성장 둔화 |

## 2. 단계별 도입 계획

### Phase 0: 소프트웨어만 (비용 0원, 즉시 시작)

설치하는 것: 딸기박사.exe + 카카오톡 채널 추가

**할 수 있는 것:**
- 스마트폰으로 딸기 사진 찍어서 카톡에 보내면 → **AI 병해 진단** (잿빛곰팡이, 흰가루병 등)
- 기상청 연동 → "내일 영하 예보, 오늘 밤 보온 주의" **자동 알림**
- 도매시장 경락가 → "현재 설향 시세 + 출하 추천" **매일 알림**
- 재배 일지 → 카톡으로 "오늘 농약 살포함" 보내면 자동 기록
- "오늘 할일" → 생육 단계 + 날씨 보고 작업 추천

**이것만으로 병충해 1순위 문제에 바로 대응 가능.**

## 3. 설치 순서 (농가에서 직접 할 수 있는 수준)

### Phase 0 (오늘 당장)

```
1. 개발자가 딸기박사.exe 빌드
2. 농가 PC에 USB로 복사 → 더블클릭 설치
3. 카카오톡에서 "딸기박사" 채널 추가
4. 딸기 사진 찍어서 카톡에 보내기 → AI 진단 테스트
5. 끝
```

### Phase 1 (부품 도착 후, 약 1~2시간/동)

```
1. ESP32에 펌웨어 업로드 (개발자가 해줄 수도, USB 꽂고 버튼 하나)
2. 방수케이스에 ESP32 + 브레드보드 조립
3. 센서 케이블 연결 (점퍼선 꽂기만 하면 됨)
4. 하우스 내부에 센서 배치 (위치 가이드대로)
5. USB 전원 연결
6. 카톡에 "상태" 입력 → 데이터 올라오면 성공
```

### Phase 2 (전기 배선 주의)

```
1. 릴레이 모듈을 ESP32에 연결 (점퍼선)
2. 기존 환풍기/커튼 모터 전원선을 릴레이 경유로 변경
   ⚠️ AC 220V 부분은 전기기사 도움 필요
3. 수위 센서를 배수구 근처에 설치
4. 카톡에서 "환풍기 켜" → 실제로 켜지면 성공
```

## 4. 기대 효과

| 문제 | 현재 | BerryDoctor 적용 후 |
|------|------|-------------------|
| 병충해 발견 | 눈으로 보고 발견 (이미 퍼진 후) | **AI 조기 감지 1~3일 빠름** |
| 난방비 | 경험으로 가동 판단 | **기상 예측 연동 → 15~20% 절감** |
| 양액/관비 | 감으로 배합 | **토양수분 데이터 기반 → 비료비 20~30% 절감** |
| 광량 | 확인 안 함 | **조도 센서 → 보광 시점 알림** |
| 침수 | 비 올 때 직접 가서 확인 | **수위 센서 → 즉시 카톡 알림** |
| 하우스 방문 | 하루 3~5회 | **1회 이하 (원격 확인)** |
| 출하 타이밍 | 중간상인 의존 | **경락가 예측 → 최적 출하일 추천** |
