## 1. 시작하기 전에

### 필요한 도구

- 드라이버 세트 (십자, 일자)
- 니퍼 또는 가위 (선 자를 때)
- 전기 테이프 또는 수축 튜브
- (선택) 납땜 인두 + 납 (영구 설치 시)

### 안전 수칙

- **ESP32와 센서는 5V/3.3V 저전압**이므로 감전 위험은 거의 없습니다.
- **릴레이로 220V 기기(환풍기, 펌프 등)를 제어할 때는 반드시 전원을 끄고 작업하세요.**
- 220V 배선에 자신이 없으면, 릴레이까지만 연결하고 220V 연결은 전기 기사에게 맡기세요.
- 하우스 내부는 습하므로 방수 케이스를 반드시 사용하세요.

## 2. ESP32 핀 배치

딸기박사 펌웨어가 사용하는 핀 번호입니다 (`firmware/src/config.h` 기준):

```
ESP32 DevKit V1 핀 배치 (위에서 내려다본 모습)

                    USB 포트
                  ┌──────────┐
            3V3  ─┤ 1     30 ├─ VIN (5V)
           GND  ─┤ 2     29 ├─ GND
           D15  ─┤ 3     28 ├─ D13 ← 릴레이: 커튼
            D2  ─┤ 4     27 ├─ D12 ← 릴레이: 환풍기
            D4  ─┤ 5     26 ├─ D14 ← 릴레이: 보광등
           D16  ─┤ 6     25 ├─ D27 ← 릴레이: CO2
           D17  ─┤ 7     24 ├─ D26 ← 펌프: 배수
            D5  ─┤ 8     23 ├─ D25 ← 펌프: 관수
           D18  ─┤ 9     22 ├─ D33 ← 펌프: B액
           D19  ─┤10     21 ├─ D32 ← 펌프: A액
           D21  ─┤11     20 ├─ D35 ← 아날로그 입력
            RX  ─┤12     19 ├─ D34 ← 아날로그 입력
            TX  ─┤13     18 ├─ VN (D39)
           D22  ─┤14     17 ├─ VP (D36)
           D23  ─┤15     16 ├─ EN
                  └──────────┘
```

### 핀 할당표

| ESP32 핀 | 연결 대상 | 용도 | 프로토콜 |
|----------|-----------|------|----------|
| **GPIO 4** | DHT22 DATA | 실내 온습도 | 1-Wire(DHT) |
| **GPIO 16** | DS18B20 DATA (외기 + 토양) | 외기/토양 온도 | 1-Wire |
| **GPIO 21** | BH1750 SDA | 조도 (I2C) | I2C |
| **GPIO 22** | BH1750 SCL | 조도 (I2C) | I2C |
| **GPIO 34** | 토양 수분 센서 #1 | 토양 수분 | 아날로그 |
| **GPIO 35** | 토양 수분 센서 #2 | 토양 수분 | 아날로그 |
| **GPIO 36 (VP)** | 엽면 습윤 센서 | 잎 젖음 | 아날로그 |
| **GPIO 39 (VN)** | 수위 센서 | 침수 감지 | 아날로그 |
| **GPIO 17** | MH-Z19B TX (→ ESP RX) | CO2 | UART |
| **GPIO 5** | MH-Z19B RX (→ ESP TX) | CO2 | UART |
| **GPIO 12** | 릴레이 CH1 | 환풍기 | 디지털 |
| **GPIO 13** | 릴레이 CH2 | 커튼 모터 | 디지털 |
| **GPIO 14** | 릴레이 CH3 | 보광등 | 디지털 |
| **GPIO 27** | 릴레이 CH4 | CO2 밸브 | 디지털 |
| **GPIO 25** | 릴레이 CH5 | 관수 펌프 | 디지털 |
| **GPIO 26** | 릴레이 CH6 | 배수 펌프 | 디지털 |
| **GPIO 32** | 릴레이 CH7 | 양액 A액 펌프 | 디지털 |
| **GPIO 33** | 릴레이 CH8 | 양액 B액 펌프 | 디지털 |

## 3. 최소 구성: 온습도 센서만 연결

가장 간단한 연결입니다. 15분이면 됩니다.

### 준비물

- ESP32 DevKit V1
- DHT22 온습도 센서 모듈 (3핀 모듈)
- 점퍼선 3개 (F-M)
- USB 케이블

### 연결 방법

```
DHT22 모듈 (앞면에서 봤을 때, 왼쪽부터)

 ┌─────────────┐
 │   DHT22     │
 │  (격자무늬)  │
 └──┬──┬──┬────┘
    │  │  │
   VCC DATA GND
    │  │    │
    │  │    │
    ▼  ▼    ▼
   3V3 D4  GND    ← ESP32 핀에 연결
```

| DHT22 핀 | 선 색상 (일반적) | ESP32 핀 |
|----------|------------------|----------|
| VCC (왼쪽) | 빨간색 | **3V3** |
| DATA (가운데) | 노란색 또는 흰색 | **GPIO 4** |
| GND (오른쪽) | 검은색 | **GND** |

> 3핀 모듈에는 풀업 저항이 이미 내장되어 있습니다. 4핀(모듈 아닌 부품)을 샀다면 DATA와 VCC 사이에 10K옴 저항을 연결하세요.

### 연결 확인

1. USB 케이블로 ESP32를 PC에 연결
2. 시리얼 모니터에서 온습도 값이 출력되는지 확인

## 4. 기본 구성: 센서 전체 연결

### 배선도

```
                         ESP32 DevKit V1
                       ┌─────────────────┐
                       │                 │
  [DHT22]──DATA────────┤ GPIO 4          │
  [DHT22]──VCC─────────┤ 3V3             │
  [DHT22]──GND─────────┤ GND             │
                       │                 │
  [DS18B20 x3]──DATA───┤ GPIO 16         │──┐
  [DS18B20 x3]──VCC────┤ 3V3             │  ├─ 4.7K 저항
  [DS18B20 x3]──GND────┤ GND             │──┘ (DATA↔VCC)
                       │                 │
  [BH1750]──SDA────────┤ GPIO 21 (SDA)   │
  [BH1750]──SCL────────┤ GPIO 22 (SCL)   │
  [BH1750]──VCC────────┤ 3V3             │
  [BH1750]──GND────────┤ GND             │
                       │                 │
  [토양수분1]──AOUT────┤ GPIO 34         │
  [토양수분1]──VCC─────┤ 3V3             │
  [토양수분1]──GND─────┤ GND             │
                       │                 │
  [토양수분2]──AOUT────┤ GPIO 35         │
  [토양수분2]──VCC─────┤ 3V3             │
  [토양수분2]──GND─────┤ GND             │
                       │                 │
  [엽면습윤]──AOUT─────┤ GPIO 36 (VP)    │
  [엽면습윤]──VCC──────┤ 3V3             │
  [엽면습윤]──GND──────┤ GND             │
                       │                 │
  [수위]──AOUT─────────┤ GPIO 39 (VN)    │
  [수위]──VCC──────────┤ 3V3             │
  [수위]──GND──────────┤ GND             │
                       │                 │
                       └─────────────────┘
```

### DS18B20 여러 개 연결 (1-Wire 버스)

DS18B20은 같은 DATA 핀 하나에 여러 개를 병렬로 연결할 수 있습니다:

```
            4.7K옴
  3V3 ──────┤저항├────┬─── GPIO 16
                     │
           ┌─────────┤
  DS18B20  │  DATA───┤      외기 온도 (하우스 밖)
  (외기)   │  VCC────┤─── 3V3
           │  GND────┤─── GND
           └─────────┤
                     │
           ┌─────────┤
  DS18B20  │  DATA───┤      토양 온도 (땅속 10cm)
  (토양)   │  VCC────┤─── 3V3
           │  GND────┤─── GND
           └─────────┤
                     │
           ┌─────────┤
  DS18B20  │  DATA───┤      양액 온도 (탱크 안)
  (양액)   │  VCC────┤─── 3V3
           │  GND────┤─── GND
           └─────────┘
```

> 1-Wire 프로토콜은 각 센서가 고유 주소를 가지고 있어서, 선 하나에 최대 수십 개까지 연결 가능합니다.
> 풀업 저항(4.7K옴)은 전체에 **1개만** 연결하면 됩니다.

### BH1750 (I2C) 연결 상세

```
  BH1750 모듈       ESP32
  ┌──────────┐
  │ VCC  ────┼──── 3V3
  │ GND  ────┼──── GND
  │ SDA  ────┼──── GPIO 21
  │ SCL  ────┼──── GPIO 22
  │ ADDR ────┼──── GND (주소 0x23) 또는 3V3 (주소 0x5C)
  └──────────┘
```

> ADDR 핀을 GND에 연결하면 기본 주소(0x23)로 동작합니다.
> 대부분의 모듈은 ADDR이 이미 GND로 연결되어 있습니다.

## 5. 릴레이 연결 (자동 제어)

### 릴레이 모듈 ↔ ESP32 연결

```
  8채널 릴레이 모듈            ESP32
  ┌──────────────────┐
  │ VCC ─────────────┼──── 5V (VIN)
  │ GND ─────────────┼──── GND
  │ IN1 ─────────────┼──── GPIO 12  (환풍기)
  │ IN2 ─────────────┼──── GPIO 13  (커튼)
  │ IN3 ─────────────┼──── GPIO 14  (보광등)
  │ IN4 ─────────────┼──── GPIO 27  (CO2)
  │ IN5 ─────────────┼──── GPIO 25  (관수 펌프)
  │ IN6 ─────────────┼──── GPIO 26  (배수 펌프)
  │ IN7 ─────────────┼──── GPIO 32  (양액 A액)
  │ IN8 ─────────────┼──── GPIO 33  (양액 B액)
  └──────────────────┘
```

> 릴레이 모듈은 **5V 전원**이 필요합니다. ESP32의 **VIN** 핀에서 가져오세요.
> IN 핀은 LOW일 때 릴레이가 켜지는 **Active LOW** 방식이 일반적입니다 (펌웨어에서 처리됨).

### 릴레이 ↔ 220V 기기 연결

```
  릴레이 CH1 (환풍기 예시)
  ┌─────────────┐
  │ COM (공통)  ├────── 220V 전원 한쪽 선
  │ NO (상시열림)├────── 환풍기 한쪽 선
  │ NC (상시닫힘)├────── (사용 안 함)
  └─────────────┘

  환풍기의 나머지 선은 220V 전원의 나머지 선에 직접 연결
```

> **COM**: Common (공통) - 전원 입력
> **NO**: Normally Open - 평소 끊김, 릴레이 켜지면 연결됨
> **NC**: Normally Closed - 평소 연결됨, 릴레이 켜지면 끊김
>
> 딸기박사는 **NO**를 사용합니다 (필요할 때만 기기를 켬).

> **경고**: 220V 작업은 반드시 **차단기를 내리고** 하세요.
> 자신이 없으면 릴레이 모듈까지만 연결하고, 220V 부분은 전기 기사에게 맡기세요.

## 6. 양액 센서 연결 (수경 재배)

### TDS/EC 센서

```
  TDS 센서 모듈       ESP32
  ┌──────────────┐
  │ VCC ─────────┼──── 3V3
  │ GND ─────────┼──── GND
  │ AOUT ────────┼──── GPIO 32 (또는 남는 ADC 핀)
  │ 프로브 ──────┼──── (양액 탱크에 침지)
  └──────────────┘
```

### pH 센서

```
  PH-4502C 모듈       ESP32
  ┌──────────────┐
  │ V+ ──────────┼──── 5V (VIN)
  │ GND ─────────┼──── GND
  │ Po ──────────┼──── GPIO 33 (또는 남는 ADC 핀)
  │ BNC 커넥터 ──┼──── pH 유리 프로브 연결
  └──────────────┘
```

> pH 센서는 **5V**가 필요합니다 (3.3V에서는 정확도가 떨어짐).
> 캘리브레이션: pH 4.0, 7.0 표준액에 담가 보정 후 사용하세요.

## 7. 펌웨어 업로드

### PlatformIO 설치

1. [VS Code](https://code.visualstudio.com/) 설치
2. VS Code 확장에서 "PlatformIO IDE" 검색 → 설치
3. VS Code 재시작

### WiFi/MQTT 설정

`firmware/src/config.h`를 열어 수정합니다:

```cpp
static constexpr char WIFI_SSID[]     = "우리집WiFi";        // WiFi 이름
static constexpr char WIFI_PASSWORD[] = "wifi비밀번호";       // WiFi 비밀번호
static constexpr char MQTT_HOST[]     = "192.168.0.10";      // 딸기박사 PC의 IP
static constexpr uint16_t MQTT_PORT   = 1883;
static constexpr uint8_t HOUSE_ID     = 1;                   // 하우스 번호
```

> **MQTT_HOST**: 딸기박사가 실행 중인 PC의 IP 주소입니다.
> PC에서 `ipconfig` (Windows) 명령으로 확인하세요.
> 예: `IPv4 주소 . . . . . . . . : 192.168.0.10`

### 업로드

1. ESP32를 USB로 PC에 연결
2. VS Code에서 `firmware/` 폴더 열기
3. 하단 상태바의 **→ (Upload)** 버튼 클릭
4. 업로드 완료 후 시리얼 모니터에서 센서 데이터 확인

> 처음 업로드할 때 USB 드라이버 설치가 필요할 수 있습니다.
> CP2102 또는 CH340 드라이버를 검색해서 설치하세요.

## 8. 하우스 내 설치 위치

```
  하우스 단면도 (딸기 고설 베드 기준)

  ┌───────────────────────────────────────────┐ ← 지붕
  │                                           │
  │    [BH1750]                               │ ← 천장 가까이 (차광 없는 곳)
  │     조도센서                               │
  │                                           │
  │  [DHT22]          [DS18B20 외기]           │ ← 작물 높이 (베드 위 30cm)
  │  온습도           하우스 밖 처마에           │
  │                                           │
  │  ┌─베드─────────────────────────베드─┐     │
  │  │ [토양수분1]  [엽면습윤]  [토양수분2]│     │ ← 베드 안 / 잎 높이
  │  │ (왼쪽 끝)    (잎 사이)   (오른쪽 끝)│     │
  │  └──────────────────────────────────┘     │
  │  [DS18B20 토양]                            │ ← 토양 속 10cm
  │                                           │
  │  [수위센서]                    [방수박스]    │ ← 바닥 / 배수로
  │  (배수로)                     ESP32+릴레이  │
  │                                           │
  └───────────────────────────────────────────┘
```

### 설치 팁

| 센서 | 위치 | 주의사항 |
|------|------|---------|
| DHT22 | 베드 위 30cm, 직사광선 피해서 | 통풍 되는 곳. 비닐 바로 아래 X |
| DS18B20 (외기) | 하우스 밖 처마 밑 | 비는 맞아도 됨 (방수). 직사광선 피해서 |
| DS18B20 (토양) | 흙 속 5~10cm | 프로브만 묻고 케이블은 밖으로 |
| 토양 수분 | 베드 양쪽 끝, 흙에 꽂기 | 코팅 부분만 묻히게 (전자부품 부분은 밖으로) |
| BH1750 | 천장 가까이, 위를 향하게 | 비닐에 물방울 맺히는 곳 피해서 |
| 엽면 습윤 | 딸기 잎 높이, 수평으로 | 잎이 닿을 정도로 가깝게 |
| 수위 | 배수로 바닥 | 하우스에서 가장 낮은 곳 |
| ESP32 + 릴레이 | 방수 박스 안, 벽면 | 물 튀지 않는 곳, 전원 가까운 곳 |

### 케이블 정리

- 센서에서 ESP32까지 거리가 멀면 (5m 이상) 연장 케이블을 사용하세요
- DS18B20은 케이블 20m까지 가능 (풀업 저항을 2.2K옴으로 낮추면 더 안정)
- I2C(BH1750)는 케이블 3m 이내 권장
- 모든 케이블은 벽면이나 파이프를 따라 정리하고, 케이블 타이로 고정

## 9. 문제 해결

### ESP32가 PC에 인식되지 않아요

1. USB 케이블이 "데이터 케이블"인지 확인 (충전 전용은 안 됨)
2. USB 드라이버 설치: "CP2102 드라이버" 또는 "CH340 드라이버" 검색
3. 다른 USB 포트에 꽂아 보세요

### 센서 값이 0이나 -999로 나와요

1. 선이 제대로 꽂혀 있는지 확인 (브레드보드 접촉 불량이 가장 흔함)
2. VCC와 GND가 바뀌지 않았는지 확인
3. DS18B20: 4.7K옴 풀업 저항이 연결되어 있는지 확인
4. 아날로그 센서: ESP32 GPIO 34, 35, 36, 39만 ADC 입력 가능 (다른 핀은 안 됨)

### WiFi에 연결이 안 돼요

1. `config.h`의 SSID와 비밀번호를 다시 확인 (대소문자, 공백 주의)
2. 2.4GHz WiFi인지 확인 (ESP32는 5GHz를 지원하지 않음)
3. 공유기와 ESP32 사이 거리가 너무 멀지 않은지 확인

### 딸기박사에 센서 데이터가 안 올라와요

1. ESP32 시리얼 모니터에서 MQTT 연결 성공 메시지 확인
2. `config.h`의 `MQTT_HOST`가 딸기박사 PC의 IP와 일치하는지 확인
3. 딸기박사 PC에서 방화벽이 1883 포트를 막고 있지 않은지 확인
4. 딸기박사 대시보드에서 센서 데이터가 표시되는지 확인

### 릴레이가 동작하지 않아요

1. 릴레이 모듈에 5V 전원(VIN)이 들어가는지 확인 (LED 점등 확인)
2. IN 핀 연결 확인
3. 릴레이 동작 소리가 나는지 귀 가까이 대고 확인 ("딸깍")
4. 릴레이 동작은 되는데 기기가 안 켜지면: COM-NO 연결 확인
