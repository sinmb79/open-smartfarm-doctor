# BerryDoctor 하드웨어 부품 목록 (BOM)

> 딸기 하우스 1동 기준, 누구나 구할 수 있는 저렴한 부품으로 구성했습니다.
> 전자공학 전공이 아니어도 따라할 수 있도록 제품명, 구매처, 용도를 모두 적었습니다.

---

## 전체 비용 요약

| 구성 | 예상 비용 | 비고 |
|------|-----------|------|
| **최소 구성** (온습도 + WiFi만) | 약 15,000원 | 소프트웨어 기능 체험용 |
| **기본 구성** (센서 6종) | 약 45,000~60,000원 | 일반 농가 권장 |
| **풀 구성** (센서 + 릴레이 제어) | 약 80,000~120,000원 | 자동 제어까지 |
| **양액 구성** (풀 + EC/pH) | 약 150,000~200,000원 | 수경 재배 농가 |

> 가격은 2024~2025년 국내 온라인 쇼핑몰 기준입니다. 해외 직구 시 더 저렴할 수 있습니다.

## 1. 메인 보드

### ESP32 DevKit V1 (ESP-WROOM-32)

딸기박사의 센서 모듈 두뇌입니다. WiFi가 내장되어 있어서 별도 통신 모듈이 필요 없습니다.

| 항목 | 내용 |
|------|------|
| **제품명** | ESP32 DevKit V1 (ESP-WROOM-32) 또는 ESP32-WROOM-32D |
| **가격** | 5,000~8,000원 |
| **구매처** | 네이버 쇼핑에서 "ESP32 개발보드" 검색 |
| | [디바이스마트](https://www.devicemart.co.kr) - "ESP32" 검색 |
| | [엘레파츠](https://www.eleparts.co.kr) - "ESP32 DevKit" 검색 |
| | [쿠팡](https://www.coupang.com) - "ESP32 개발보드" 검색 |
| | AliExpress - "ESP32 DevKit V1" (해외직구, 3,000원대) |
| **왜 이 제품인가** | WiFi 내장, GPIO 핀 30개 이상, 5V/3.3V 모두 지원, Arduino 호환 |

> **팁**: "ESP32-S3"이나 "ESP32-C3"도 사용 가능하지만, 핀 배치가 다를 수 있습니다.
> 처음이라면 가장 흔한 "ESP32 DevKit V1"을 권장합니다.

## 2. 센서류

### 2-1. 실내 온습도 센서 (필수)

가장 중요한 센서입니다. 이것만 있어도 환기 타이밍을 잡을 수 있습니다.

| 항목 | 내용 |
|------|------|
| **제품명** | DHT22 (AM2302) 온습도 센서 모듈 |
| **가격** | 3,000~5,000원 |
| **구매처** | 디바이스마트/엘레파츠 "DHT22" 검색, 쿠팡 "DHT22 센서" |
| **측정 범위** | 온도 -40~80도C (오차 +-0.5도), 습도 0~100% (오차 +-2%) |
| **연결 핀** | 3핀 (VCC, DATA, GND) - 풀업 저항 내장 모듈 사용 권장 |
| **펌웨어 매핑** | `tempIndoor`, `humidity` |

> **대안**: SHT30 또는 SHT31 (I2C 방식, 더 정밀하지만 5,000~8,000원)
> **주의**: DHT11은 저렴하지만(1,500원) 습도 오차가 +-5%라 농업용으로 부정확합니다.

### 2-2. 외기 온도 센서

하우스 밖 온도를 측정합니다. 동해 경보에 사용됩니다.

| 항목 | 내용 |
|------|------|
| **제품명** | DS18B20 방수형 온도 센서 (스테인리스 프로브) |
| **가격** | 2,000~3,500원 |
| **구매처** | "DS18B20 방수" 검색 |
| **측정 범위** | -55~125도C (오차 +-0.5도) |
| **특징** | 방수 프로브라 비를 맞아도 됩니다. 케이블 길이 1m~3m 선택 가능 |
| **연결** | 3선 (VCC, DATA, GND) + 4.7K옴 풀업 저항 1개 필요 |
| **펌웨어 매핑** | `tempOutdoor` |

### 2-3. 토양 수분 센서

| 항목 | 내용 |
|------|------|
| **제품명** | 정전식 토양 수분 센서 v1.2 (Capacitive Soil Moisture Sensor) |
| **가격** | 1,500~3,000원 |
| **구매처** | "정전식 토양수분센서" 검색 |
| **수량** | 2개 권장 (하우스 양쪽 끝) |
| **출력** | 아날로그 (0~4095, ESP32 ADC) |
| **펌웨어 매핑** | `soilMoisture1`, `soilMoisture2` |

> **중요**: "저항식" 토양 수분 센서(2개 금속판 노출형)는 부식이 빨라 비추천합니다.
> 반드시 "정전식(Capacitive)" 제품을 사세요. 겉이 검은색 코팅으로 덮여 있습니다.

### 2-4. 토양 온도 센서

| 항목 | 내용 |
|------|------|
| **제품명** | DS18B20 방수형 (외기 온도와 같은 제품) |
| **가격** | 2,000~3,500원 |
| **용도** | 토양에 5~10cm 깊이로 삽입 |
| **펌웨어 매핑** | `soilTemp` |

> 외기 온도 센서와 같은 DS18B20을 쓰면, 데이터 핀 1개에 여러 개를 연결할 수 있습니다 (1-Wire 버스).

### 2-5. 광량 센서

| 항목 | 내용 |
|------|------|
| **제품명** | BH1750 디지털 조도 센서 모듈 |
| **가격** | 2,000~3,500원 |
| **구매처** | "BH1750" 검색 |
| **측정 범위** | 1~65,535 lux |
| **연결** | I2C (SDA, SCL, VCC, GND) - 4핀 |
| **펌웨어 매핑** | `lightLux` |

> **대안**: TSL2561 (비슷한 가격, 적외선 분리 측정 가능)

### 2-6. 엽면 습윤 센서

잎이 젖어있는 시간을 측정합니다. 잿빛곰팡이 위험도 계산에 핵심입니다.

| 항목 | 내용 |
|------|------|
| **제품명** | 빗물/물방울 감지 센서 모듈 (Raindrop Sensor) |
| **가격** | 1,000~2,000원 |
| **구매처** | "빗물감지센서" 또는 "water detection sensor" 검색 |
| **출력** | 아날로그 + 디지털 (모듈에 비교기 내장) |
| **설치** | 딸기 잎 높이에 수평으로 설치 |
| **펌웨어 매핑** | `leafWetness` |

### 2-7. 수위 센서 (선택)

하우스 바닥 침수 감지용입니다.

| 항목 | 내용 |
|------|------|
| **제품명** | 수위 감지 센서 모듈 (Water Level Sensor) |
| **가격** | 1,000~1,500원 |
| **구매처** | "수위센서 아두이노" 검색 |
| **출력** | 아날로그 (0~4095) |
| **설치** | 배수로 또는 하우스 바닥 낮은 곳 |

### 2-8. CO2 센서 (선택)

| 항목 | 내용 |
|------|------|
| **제품명** | MH-Z19B CO2 센서 |
| **가격** | 15,000~25,000원 |
| **구매처** | "MH-Z19B" 검색 |
| **측정 범위** | 0~5,000ppm (오차 +-50ppm) |

> 처음에는 생략하고 나중에 추가해도 됩니다.

## 3. 양액 센서 (수경 재배 전용)

토경 재배라면 이 섹션은 건너뛰세요.

### 3-1. EC 센서

| 항목 | 내용 |
|------|------|
| **제품명** | DFRobot Analog EC Meter V2 (DFR0300) 또는 TDS 미터 모듈 |
| **가격** | EC 전용: 25,000~35,000원 / TDS 모듈: 5,000~8,000원 |
| **구매처** | "EC미터 아두이노" 또는 "TDS센서 아두이노" 검색 |
| **연결** | 아날로그 (프로브 + 변환 보드) |
| **펌웨어 매핑** | `solutionEc` |

> **저렴한 대안**: TDS 미터 모듈(5,000원)을 사서 TDS 값을 EC로 환산할 수 있습니다.
> EC = TDS / 500 (대략적 환산)

### 3-2. pH 센서

| 항목 | 내용 |
|------|------|
| **제품명** | pH 센서 모듈 (PH-4502C + 유리 프로브) |
| **가격** | 8,000~15,000원 |
| **구매처** | "pH센서 아두이노" 검색 |
| **연결** | 아날로그 (프로브 + 변환 보드) |
| **주의** | 유리 프로브는 6개월~1년마다 교체 필요 (교체 프로브 5,000원 내외) |
| **펌웨어 매핑** | `solutionPh` |

### 3-3. 양액 온도 센서

| 항목 | 내용 |
|------|------|
| **제품명** | DS18B20 방수형 (위와 동일) |
| **가격** | 2,000~3,500원 |
| **설치** | 양액 탱크에 프로브 침지 |
| **펌웨어 매핑** | `nutrientTemp` |

| **가격** | 4,000~7,000원 |
| **구매처** | "8채널 릴레이 모듈 아두이노" 검색 |
